Sobre Mim

Desenvolvedor apaixonado por tecnologia e inovação

Sou um desenvolvedor Full Stack com experiência sólida em desenvolvimento web utilizando Java com Spring Boot, Angular e React. Além disso, tenho expertise em projetos embarcados com ESP32 e Raspberry Pi.

Atualmente, trabalho como Estagiário em Desenvolvimento de Software no Superior Tribunal de Justiça (STJ), onde desenvolvo aplicações internas para gestão de riscos e compliance, utilizando Java Spring Boot no backend e Angular no frontend.

Minha paixão está em criar soluções integradas que combinam desenvolvimento web com automação, IoT e sistemas embarcados, sempre buscando entregar valor real através da tecnologia.
