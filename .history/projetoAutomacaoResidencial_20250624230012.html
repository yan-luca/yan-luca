Sobre o Projeto

Este projeto implementa um sistema completo de automação residencial usando microcontroladores ESP32 e protocolo MQTT. O sistema permite controle remoto de lâmpadas através de uma interface web integrada com Home Assistant.

A solução combina hardware embarcado, comunicação sem fio e interface web para criar uma experiência de automação residencial moderna e acessível. O sistema é escalável e pode ser expandido para controlar outros dispositivos eletrônicos.

Principais funcionalidades:

  • Controle remoto via aplicativo web
  • Comunicação MQTT para baixa latência
  • Interface Home Assistant integrada
  • Feedback em tempo real do status dos dispositivos
  • Configuração WiFi automática

Etapa 3

Programação dos Microcontroladores

Desenvolvimento do firmware em C++ para os ESP32, implementando comunicação WiFi, protocolo MQTT e controle dos relés.

main.cpp - ESP32 Firmware
#include <WiFi.h> #include <PubSubClient.h> const char* ssid = "SUA_REDE_WIFI"; const char* password = "SUA_SENHA"; const char* mqtt_server = "192.168.1.100"; WiFiClient espClient; PubSubClient client(espClient); const int relayPin = 2; bool lightState = false; void setup() { Serial.begin(115200); pinMode(relayPin, OUTPUT); setup_wifi(); client.setServer(mqtt_server, 1883); client.setCallback(callback); } void callback(char* topic, byte* payload, unsigned int length) { String message; for (int i = 0; i < length; i++) { message += (char)payload[i]; } if (String(topic) == "home/light/living_room") { if (message == "ON") { digitalWrite(relayPin, HIGH); lightState = true; } else if (message == "OFF") { digitalWrite(relayPin, LOW); lightState = false; } // Enviar status de volta client.publish("home/light/living_room/status", lightState ? "ON" : "OFF"); } }
